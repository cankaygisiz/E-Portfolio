SolarWinds Exploit Analysis using Cyber Kill Chain

Cyber Killchain Phase Description Mitigations Tools
1. Reconnaissance Gathering information about the target - Regularly update and patch systems. - Implement strict access controls. - WHOIS databases - Shodan - Google Dorks
2. Weaponization Developing or obtaining the exploit payload - Use advanced threat detection tools. - Conduct regular security awareness training. - Metasploit - VirusTotal
3. Delivery Transmitting the exploit to the target - Implement email filtering and scanning. - Use endpoint protection solutions. - Phishing simulation tools - Sandboxing tools
4. Exploitation Executing the exploit on the target system - Keep software and systems up-to-date. - Employ intrusion detection/prevention systems. - Cobalt Strike - Nessus
5. Installation Installing malware on the compromised system - Use application whitelisting. - Regularly monitor system logs. - Wireshark - Sysinternals Suite
6. Command and Control (C2) Establishing communication with attacker-controlled servers - Implement network segmentation. - Employ threat intelligence feeds. - Snort - Bro/Zeek
7. Actions on Objectives Achieving the attacker's goals - Regularly audit and monitor user accounts. - Conduct penetration testing. - PowerShell - Mimikatz
